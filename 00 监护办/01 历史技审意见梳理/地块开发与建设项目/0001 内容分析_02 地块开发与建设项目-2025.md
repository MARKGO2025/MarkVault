## 3 三、 工程特征总结

### 3.1 建筑规模与形态
- **土地面积**：从千余平方米（如徐汇天平街道地块，1521.94m²）到近20万平方米（如东方枢纽上海东站A3-01地块，192127.34m²）。
- **建筑高度**：普遍包含高层建筑，塔楼高度多在50-100米之间（10-30层），配套裙房多为1-4层。
- **地下开发**：几乎所有项目都设有地下室，普遍为1-3层，开挖深度集中在 **5m至16m** 之间。

### 3.2 基础与基坑工程（共性技术措施）

| 工程环节 | 通用技术方案 | 目的 |
| :--- | :--- | :--- |
| **建筑基础** | 地铁保护区内**强制使用非挤土桩**（主要为**钻孔灌注桩**），桩长30m-70m不等，持力层多为⑦、⑧、⑨层土。 | 避免挤土效应，控制沉降。 |
| **基坑分区** | 绝大多数项目采用**分区分块**施工，遵循“**先远后近**”（先开挖远离地铁侧）的原则。 | 化整为零，控制变形，减少对地铁的叠加影响。 |
| **基坑围护** | 近地铁侧常用**钻孔灌注桩 + 三轴搅拌桩/MJS工法桩/高压旋喷桩止水帷幕**。非地铁侧可用SMW工法。 | 提供强有力的挡土和止水屏障。 |
| **坑内加固** | 近地铁侧采用**水泥土搅拌桩（三轴/双轴）进行裙边或满堂加固**，强度要求qu≥0.8MPa。 | 提高坑底土体稳定性，减少围护位移和坑底隆起。 |
| **支撑体系** | 多采用**钢筋混凝土支撑**（对撑+角撑）或结合**钢支撑**，并强调“先撑后挖”。 | 控制围护结构变形，确保基坑稳定。 |

### 3.3 水文地质处理
- **降水**：大部分项目仅需进行**疏干降水**。当开挖较深涉及承压水层时，采用**隔断（如MJS）** 或**降压**措施。
- **防水**：邻近地铁侧的地下室普遍要求按 **“一级防水标准”** 设计，以结构自防水为主。

## 4 四、 核心特征与趋势

1.  **高强度开发**：项目普遍呈现“高层塔楼+多层地下室”的特征，土地利用率高。
2.  **TOD模式普遍化**：大量项目与地铁车站“结建”或设置连通口，体现了轨道交通引导城市发展的TOD模式已成常态。
3.  **技术方案标准化**：对于邻近地铁的建设项目，一套以“钻孔灌注桩+分块开挖+强力围护止水+坑内加固+钢筋混凝土支撑”为核心的**保护性技术体系已非常成熟和标准化**。
4.  **风险管控精细化**：审查意见体现出极高的精细化程度，根据不同项目的具体位置、地铁结构现状和工程特点，提出了差异化的变形控制指标和专项保护要求，实现了“一地一策”的精准管控。
5.  **全生命周期管理**：不仅关注施工期安全，还通过产权界定、协议签订、运营管理界面划分等方式，明确了项目在整个使用周期内对轨道交通设施的责任与义务。

综上所述，这些项目是上海在密集建成区、紧邻运营轨道交通线路环境下进行深度城市开发的典型代表，其工程概况集中反映了当前城市更新和新区建设中，为保障轨道交通“生命线”安全所必须遵循的高标准、严要求和技术复杂性。

## 1 一、总体审查结论

所有项目均**原则同意**送审作业方案，但均需在后续设计、施工中严格遵守轨道交通保护相关法规与技术标准，确保地铁结构与运营安全。

## 2 二、共同要求总结

### 2.1 法律依据
所有项目均依据以下文件进行审查：
- 《上海市轨道交通管理条例》
- 《上海市轨道交通运营安全管理办法》（2022年第62号政府令）
- 《上海市人民政府办公厅关于本市保障轨道交通安全运行的实施意见》（沪府办发〔2019〕10号）
- 《上海市人民政府关于进一步加强本市轨道交通管理的意见》（沪府发〔2012〕38号）

### 2.2 通用保护要求
- **变形控制**：地铁结构沉降/隆起、水平位移累计量一般≤10mm，部分复杂项目放宽至≤20mm。
- **收敛变形**：根据隧道初始状态分级控制（如初始变形≥6cm时控制值＜5mm）。
- **附加荷载**：地铁结构正上方及两侧一倍埋深范围内附加荷载基本为零。
- **勘察限制**：所有勘察孔须退地铁结构10m外，钻孔后需可靠回填。
- **施工监护**：施工前须办妥地铁监护手续，落实监测措施。

### 2.3 施工组织要求
- **分区施工**：基坑应分区、分块、对称、平衡、限时开挖。
- **支撑体系**：一般采用钢筋混凝土支撑或钢支撑，严禁爆破拆除。
- **降水控制**：适时适量降水，避免过量。
- **信息化施工**：实时监测，数据及时报送，24小时值班。

## 3 三、差异分析及原因

| 差异点         | 差异表现                             | 原因分析                        |
| ----------- | -------------------------------- | --------------------------- |
| **变形控制标准**  | 多数项目≤10mm，部分如张园、长宁E2-05等放宽至≤20mm | 与地铁结构距离、结构现状（如初始变形）、施工复杂度有关 |
| **桩基类型**    | 多数采用钻孔灌注桩，部分非保护区采用PHC管桩          | 保护区范围内禁用挤土桩，避免对地铁结构产生挤压影响   |
| **基坑围护方式**  | 近地铁侧常用MJS、高压旋喷桩等止水帷幕，非地铁侧可用SMW工法 | 近地铁侧对止水、变形控制要求更高            |
| **是否结建/连通** | 部分项目与地铁站连通（如虹梅街道、张园），部分仅为保护区施工   | 取决于地块规划与地铁站的功能整合需求          |
| **产权与资产管理** | 结建项目需明确产权归属、办理产证移交、签订管理协议        | 涉及地铁设施长期使用与管理责任划分           |
| **防淹与防水要求** | 近出入口/风井项目需设防淹闸槽、高差控制、一级防水        | 防止雨水倒灌、保障地铁运营安全             |
| **振动与噪音提示** | 所有项目均提示地铁运营振动噪音，需建设单位自行采取措施并告知用户 | 法律风险防范与责任界定                 |

## 4 四、典型项目差异举例

### 4.1 **张园西区（115-06、08地块）**
- **特点**：与三线换乘通道结建，结构复杂。
- **差异**：变形控制≤20mm，需代建换乘通道土建部分。
- **原因**：结构共构、功能整合，施工影响范围大。

### 4.2 **东方枢纽上海东站A3-01地块**
- **特点**：21号线隧道穿越地块下方。
- **差异**：桩基与隧道最小水平净距仅2.1m，需严格控制桩位。
- **原因**：隧道穿越地下室，施工精度要求极高。

### 4.3 **南站xh196f-11地块（临时绿化）**
- **特点**：仅涉及绿化养护。
- **差异**：无基坑、无桩基，重点控制侵限与附加荷载。
- **原因**：项目性质简单，风险点集中于机械作业与树木种植。

### 4.4 **宝山区共康社区AB-02B、B1-01A地块**
- **特点**：与18号线呼兰路站结建。
- **差异**：需签订结合建设协议，明确产权、投资、管理界面。
- **原因**：地铁附属设施位于地块内，需确保运营与维护条件。

## 5 五、结论

- 所有项目在**技术审查**上均以“原则同意”为前提，但均需在**设计深化、施工组织、监测监护**等方面严格落实地铁保护要求。
- **差异主要源于**：
  - 项目与地铁结构的相对位置关系；
  - 地铁结构现状（如初始变形、结构健康状态）；
  - 是否涉及结建、连通、产权移交等非技术因素；
  - 项目类型（住宅、商业、绿化、拆除等）及其风险点。
- **共性要求**体现了上海地铁保护的统一标准；**差异性要求**则体现了“一地一策”的精细化管控思路。
