# 基于分布式光纤（DAS）的地铁隧道外界入侵、轨道波磨和地下构造实时探测技术



### 0.1 **总结对比表格**

|特性|**瑞利散射**|**拉曼散射**|**布里渊散射**|
|---|---|---|---|
|**散射类型**|弹性散射|非弹性散射|非弹性散射|
|**能量交换**|无|有（与光学声子）|有（与声学声子）|
|**频移量**|零|大（约13 THz）|小（约10-11 GHz）|
|**信号强度**|最强|最弱|介于两者之间|
|**主要敏感参量**|振动、扰动、损耗|**温度**|**温度** 和 **应变**|
|**核心技术**|DVS, Φ-OTDR|DTS|BOTDR, BOTDA, BOFDA|
|**核心优势**|超高振动灵敏度|纯温度测量，不受应变干扰|长距离、同时测温和应变|
分布式光纤传感主要利用三种背向散射光：**瑞利散射、拉曼散射和布里渊散射**。温度对它们的影响方式不同。

|散射类型|对温度的敏感性|原理与表现|
|---|---|---|
|**瑞利散射**|**微弱/不直接**|瑞利散射由光纤材料密度的微观不均匀引起，其**中心频率**不随温度变化。但其**强度**会因光纤的微观变形而轻微改变，因此主要用于振动和扰动定位，一般不用于直接测温。|
|**拉曼散射**|**高度敏感**|拉曼散射分为**斯托克斯光**和**反斯托克斯光**。**反斯托克斯光的强度对温度高度敏感**，而斯托克斯光对温度基本不敏感。通过计算两者强度的比值，可以直接、精确地解调出光纤各点的温度值。**这是分布式温度传感系统的核心原理。**|
|**布里渊散射**|**高度敏感（并与应变耦合）**|布里渊散射的**频移量**和**功率**都会受到温度和应变的变化的影响。  <br>1. **频移变化：** 温度升高/应变增大，都会导致布里渊散射的中心频率发生漂移。  <br>2. **功率变化：** 温度升高/应变增大，也会导致布里渊散射的强度（功率）发生变化。  <br>**关键问题：** 你无法仅通过布里渊频移或功率的变化，来区分这种变化是由温度引起的，还是由应变引起的。这被称为 **“温度-应变交叉敏感性”**。|

**结论：**

- 如果你想**只测温度**，使用基于**拉曼散射**的分布式系统是最佳选择，因为它不受应变干扰。
    
- 如果你想**测量应变**，使用基于**布里渊散射**的系统能力最强，但你必须解决如何将温度的影响从应变中剥离出去，即**温度补偿**。

在实际工程中，例如在桥梁、管道、地铁等结构的健康监测中，**“一根参考光纤 + 一根或多根传感光纤”** 的布设方案是最为常见的。


### 0.2 **DAS 是什么？**

**DAS** 是一种利用光纤本身作为传感器，**连续、实时地探测并定位沿着光纤传播的声波或振动**的技术。

它的核心特点是：

- **分布式：** 整条光纤就是传感器，无需在特定点安装传感器。
    
- **声波/振动传感：** 它对声音和振动极其敏感。
    
- **长距离监测：** 单套系统可以监测几十甚至上百公里的光纤。
### 0.3 **DAS 的核心原理**

DAS 的技术基础是 **Φ-OTDR**，它与我们之前讨论的基于拉曼和布里渊散射的技术有本质区别。

**1. 依赖的物理效应：瑞利散射**  
DAS 主要依赖于**瑞利散射**。与拉曼和布里渊散射不同，瑞利散射是弹性散射，其**频率不会改变**。

**2. 核心测量参量：相位干涉**  
DAS 的“神来之笔”在于，它不关心瑞利散射光的强度变化，而是极其精确地测量其**相位**变化。


**工作原理简化流程：**

1. **发射极相干光：** DAS 设备向光纤中注入一个功率很高、相干性极好的激光脉冲。
    
2. **形成随机散斑图：** 这个脉冲在光纤中传播时，沿途无数个散射点都会产生瑞利散射光并传回接收端。这些后向散射光由于来自光纤中不同深度的随机散射点，会相互干涉，形成一个稳定的、但强度随机分布的**后向散射曲线**，看起来像许多随机起伏的“散斑”。
    
3. **监听相位变化：** 当光纤沿线某处受到外部声波/振动（如脚步声、车辆经过、挖掘声）的干扰时，该处光纤的**长度和折射率会发生微小的变化**（即发生微应变）。
    
4. **探测变化：** 这种微应变会改变经过该点的激光脉冲的**相位**。当脉冲继续向前传播并产生后向散射时，其返回信号的**干涉图案**就会发生改变。
    
5. **定位与重构：** 系统通过比较新测得的后向散射曲线与无干扰时的“背景”曲线，通过**相干检测**技术，可以极其灵敏地探测到这种干涉图案的变化。通过OTDR的时域定位原理，就能精确计算出振动事件发生的位置。
    

**简单比喻：**  
把整条光纤想象成一根**极其灵敏的“琴弦”**。外部任何微小的振动（声波）都会像手指拨动这根琴弦一样，使其产生微小的形变。DAS系统就像一個拥有 **“绝对音感”** 的监听器，不仅能听出琴弦的哪个位置被拨动了，还能分析出拨动的力度和频率。


### 0.4 **DAS 的主要特点**

| 优势                                                          | 挑战/局限                                                                                        |
| ----------------------------------------------------------- | -------------------------------------------------------------------------------------------- |
| ✔ **超高灵敏度：** 能探测到极其微弱的振动，甚至包括人的脚步声、谈话声。                     | ❌ **只能定性，难以定量：** DAS能精确告诉你 **“何时、何地、发生了什么频率的振动”** ，但很难直接告诉你 **“振动的绝对物理量是多少”** （比如具体变形了多少毫米）。 |
| ✔ **真正的分布式测量：** 每米甚至更短的距离就是一个传感单元，无盲区。                      | ❌ **易受干扰：** 由于灵敏度太高，环境中的非目标振动（如风声、背景交通噪）也会被记录，需要强大的信号处理算法来识别目标事件。                            |
| ✔ **长距离实时监测：** 监测距离可达50公里以上，并能实现7x24小时不间断实时监控。              | ❌ **定位假事件：** 需要人工智能和机器学习算法来有效区分真正的威胁（如入侵）和无关干扰（如动物跑过）。                                       |
| ✔ **仅需普通通信光缆：** 在很多应用中，可以直接利用现有的通信光缆进行监测，无需专门铺设传感光缆，大大降低成本。 | ❌ **解释难度高：** 数据量巨大，需要专业知识和算法对振动信号源进行解释和分类。                                                   |
| ✔ **无源、抗电磁干扰：** 和所有光纤传感技术一样，本质安全，适用于易燃易爆等恶劣环境。              |                                                                                              |
