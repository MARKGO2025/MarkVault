# 香港稳定币产业生态再梳理 - 报告摘要

## 报告信息
- **来源**：上海证券
- **作者**：袁威津
- **日期**：2025-08-31

---

## 一、 主要观点

### 1.1 国际经验参考
- **美国《GENIUS法案》要点**：
  - 发行需获监管批准。
  - 要求稳定币以**1:1的比例与美元挂钩**。
  - 进行月度审查。
  - 建立反洗钱与经济制裁合规方案。
  - 对资产托管方有明确要求。
- **Circle（USDC发行方）商业模式分析（基于2024年财报）**：
  - **核心收入**：**储备资产收益**占总收入的99.10%。
  - **主要成本**：**分销支出**占总营收比重达60.30%，表明其向交易平台支付了高额的引流成本。
  - **资产托管**：与**贝莱德**成立Circle储备基金，并通过**纽约梅隆银行**进行托管。

### 1.2 香港监管框架与发展现状
- **监管进程**：
  - 2024年3月：金管局推出稳定币发行“沙盒”。
  - 2025年8月：正式实施 **《持牌稳定币发行人监管指引》** ，重点强调 **“流动资产”与“风险防范”** 要求。
- **交易平台牌照**：
  - 香港证监会已发放**11张**虚拟资产交易平台牌照。
  - **获牌机构**：包括OSL数字证券有限公司、富途控股证券旗下公司等。
  - **申请中机构**：包括胜利证券和bixin交易所所属公司。

### 1.3 香港稳定币应用场景探索
- **Evergreen项目（2021年）**：协助政府成功发行两批**代币化绿色债券**。
- **Ensemble项目“沙盒”（2024年8月启动）**：
  - **首阶段用例**：
    1.  固定收益和投资基金
    2.  流动性管理
    3.  绿色和可持续金融
    4.  贸易和供应链融资
  - **参与机构**：中银香港、渣打银行、HashKey、蚂蚁数科等。

### 1.4 香港稳定币产业生态环节
基于国际经验与本地实践，报告梳理出香港稳定币生态的五大核心环节：
1.  **稳定币发行方**
2.  **资产托管机构**
3.  **数字货币交易所**
4.  **稳定币各应用端**
5.  **技术实现方**（区块链技术、网络安全）
